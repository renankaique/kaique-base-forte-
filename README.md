# Controle_de_estoque_

Este projeto é um sistema simples criado para registrar produtos, controlar quantidades disponíveis e identificar itens que estão em falta no estoque. Ele permite cadastrar, editar e excluir produtos, além de visualizar facilmente o que precisa ser reposto. O sistema foi desenvolvido usando HTML e CSS no front-end para montar a interface e deixá-la organizada e intuitiva, enquanto o back-end foi feito em Python com Flask, responsável por receber os dados, validar informações, salvar no banco e retornar os resultados para o usuário. Todas as informações são armazenadas em um banco de dados SQLite, que guarda produtos, quantidades e preços de forma permanente. O objetivo é oferecer uma ferramenta prática e direta para auxiliar no controle básico de estoque.

Claro! Aqui vai uma explicação **simples, clara e resumida** do que cada ferramenta utilizada no seu projeto faz:

---

### **HTML**

HTML é a linguagem responsável por montar a estrutura das páginas. Ele cria os campos, tabelas, botões e textos que aparecem para o usuário. É como o “esqueleto” do site.

---

### **CSS**

CSS é a linguagem usada para deixar o visual bonito. Ele adiciona cores, tamanhos, fontes, espaçamentos e organiza o layout da página. É o “estilo” do site.

---

### **Python**

Python é a linguagem de programação usada no back-end. Ele faz toda a lógica do sistema: recebe dados, processa, faz cálculos, valida informações e se comunica com o banco de dados.

---

### **Flask**

Flask é um framework simples feito para criar sites usando Python. Ele conecta o front-end com o back-end, permitindo que o HTML envie dados para o Python e receba respostas. É ele que cria as rotas, como “/cadastrar” ou “/excluir”.

---

### **SQLite**

SQLite é o banco de dados do sistema. Ele guarda todos os produtos cadastrados, suas quantidades e preços de forma permanente. Funciona como uma “caixa” onde todas as informações ficam salvas.
